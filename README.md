# FitScope Global

Static GitHub Pages build for the Outbound Apparel Fit & Review Intelligence dashboard.

## Public Dashboard

Open the deployed dashboard here:

https://yuyangchen11.github.io/fitscope-global-dashboard/

This is the link to share with an advisor. It works from any network and does not require the local computer to be running.

## Local Development Only

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173/site/
```

The localhost link only works on the computer running the local server.

## GitHub Pages

This folder is intentionally small. It contains only the dashboard and the processed CSV files needed by the web page. Raw datasets are not included.
