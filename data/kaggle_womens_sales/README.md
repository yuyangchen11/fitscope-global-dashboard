# Kaggle Women Clothing Ecommerce Sales

Source: https://www.kaggle.com/datasets/shilongzhuang/-women-clothing-ecommerce-sales-data

Purpose in dashboard:
- Validates that real order-style CSV data can be staged in the project.
- Fields available: `order_id`, `order_date`, `sku`, `color`, `size`, `unit_price`, `quantity`, `revenue`.
- Processed outputs summarize daily sales, SKU revenue, and size mix.

Current limitation:
- The raw dataset has no product category field, so it is not yet used as the primary category KPI layer.
